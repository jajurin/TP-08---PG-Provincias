PRIMERO INSTALAR DEPENDENCIAS

EN LA SQL QUERY EJECUTAR LA QUUERY, PRIMERO EL CREATE LUEGO LO DEMAS. PEGAR LOS DATOS CON INSERT Y NO COPY.

LUEGO HACER NODE INDEX.JS
